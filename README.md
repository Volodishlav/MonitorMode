# 📡 MonitorMode

A simple Python CLI tool to enable and manage wireless monitor mode using airmon-ng.

---

### 🧠 Features

- 📶 Enable monitor mode on a selected network interface  
- 🛑 Stop monitor mode with a single key press  
- 🔁 Restart monitoring quickly  
- 🔄 Switch between interfaces easily  
- ⚡ Lightweight and minimal setup  

---

### ⚙️ Requirements

- Python 3.x  
- Aircrack-ng installed  
- Root privileges  

---

### ⚠️ Disclaimer

This tool is intended for educational purposes and authorized wireless testing only.  
Do not use it on networks without permission.
